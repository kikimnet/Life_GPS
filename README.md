# LifeGPS

LifeGPS is a minimalist, execution-focused productivity web application designed to help users transform their intentions into concrete actions and results, inspired by the "Goal – Plan – System" method.

## Features
- **Dashboard**: Daily overview of execution, habits, and focus.
- **Vision**: Annual vision planning across different life domains.
- **Goals & Plans**: Define clear goals and break them down into actionable milestones.
- **Systems (Habits)**: recurring actions designed to automate success, featuring a "Minimum Viable Day" concept.
- **Habit Tracker**: Track daily consistency to build long-term streaks.
- **Weekly & Monthly Reviews**: Guided reflections to course-correct and align actions with the broader vision.
- **Journal**: A personal knowledge base for logging lessons learned.

## Design
The UI was meticulously designed in Google Stitch using a Notion/Linear-inspired dark aesthetic (`#0e0e10` background, `#1c1c1f` surfaces), focusing on flat design, typography, and clear visual hierarchy.
